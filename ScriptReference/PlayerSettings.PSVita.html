PSVita

class in UnityEditor

Switch to Manual

Description

特定于 PS Vita 的播放器设置。

Static Variables

acquireBGM获取 PS Vita 背景音乐。
appVersionPS Vita 应用程序版本。
category软件包构建类别。
contentID应用程序内容 ID。
drmTypePS Vita DRM 类型。
enterButtonAssignment指定是将圆形还是十字形用作默认输入按钮。
healthWarning指定是否将生命值警告添加到软件手册。
infoBarColor指定 PS Vita 信息栏的颜色，true 表示白色，false 表示黑色。
infoBarOnStartup指定是否在应用程序启动时显示 PS Vita 信息栏。
keystoneFileKeystone 文件。
liveAreaBackroundPathPS Vita 实时区域背景图像。
liveAreaGatePathPS Vita 实时区域门图像。
liveAreaPathPS Vita 实时区域路径。
liveAreaTrialPathPS Vita 实时区域试验路径。
manualPathPS Vita 软件手册。
masterVersionPS Vita 内容主版本。
mediaCapacity此版本应始终为 01.00。
memoryExpansionModePS Vita 内存扩展模式。
npAgeRatingPSN 年龄分级。
npCommsPassphrasePS Vita NP 口令短语。
npCommsSigPS Vita NP 签名。
npCommunicationsIDPS Vita NP 通信 ID。
npSupportGBMorGJP支持游戏启动消息或者加入游戏的存在状态。
npTitleDatPathPS Vita NP 游戏数据文件。
npTrophyPackPath指定从何处复制奖品包的路径。
packagePassword如果要访问软件包内容，请使用 32 字符密码。
paramSfxPath指定从何处复制软件包参数文件 (param.sfx) 的路径。
parentalLevelPS Vita 家长级别。
patchChangeInfoPath用于累计补丁包。
patchOriginalPackage用于构建累计补丁包。
powerModePS Vita 功耗模式。
saveDataQuota保存数据配额。
scriptOptimizationLevel指示 IL2CPP 脚本的优化级别，范围为 0 到 2。0 表示无优化，而 2 表示最大优化。
shortTitle应用程序简称。
storageTypePS Vita 媒体类型。
tvBootModePS Vita TV 启动模式。
tvDisableEmuPS Vita TV 禁用仿真标志。
upgradable指示这是可升级（试验）类型的应用程序，可通过购买升级转换为完整的应用程序。
useLibLocationSCE 在 SDK 3.570 中删除了对 PS Vita 位置库的支持。
