LocalNotification

class in UnityEngine.iOS

Switch to Manual

Description

iOS.LocalNotification 是 Apple UIKit 框架中 UILocalNotification 类的封装器，仅适用于 iPhone/iPad/iPod Touch。

它表示应用程序可安排在特定日期和时间向其用户呈现的通知。 操作系统负责在指定时间发送通知。 本地通知与远程通知类似，但它在本地安排和发送，不需要连接到远程服务器。

使用 NotificationServices 类安排和处理本地通知。

Static Variables

defaultSoundName默认系统声音。（只读）

Variables

alertAction操作按钮或滑动条的名称。
alertBody通知警报中显示的消息。
alertLaunchImage当用户点击操作按钮时，标识用作启动图像的图像。
applicationIconBadgeNumber要显示为应用程序的图标标示的数字。
fireDate系统应发送通知的日期和时间。
hasAction用于控制警报操作是否可见的布尔值。
repeatCalendar用于重新安排通知的日历类型（公历、中国农历等）。
repeatInterval用于重新安排通知的日历间隔。
soundName显示警报时要播放的声音文件的名称。
timeZone发送通知时所用的时区。
userInfo用于将自定义信息传递给所通知应用程序的字典。

Constructors

LocalNotification创建新的本地通知。
